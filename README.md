# dataToMarkdown

## Install

```bash
npm install --save datatomarkdown
```

## Usage

```jsx
import React, { Component } from 'react'

import DataToMarkdown from 'datatomarkdown'

class Example extends Component {
  render() {
    return <DataToMarkdown />
  }
}
```
